# Лабуть Эвелина Юрьевна БПИ218
## Условие (Вариант 6)
Племя из N дикарей ест вместе из большого горшка, который вмещает M кусков тушеного миссионера. Когда дикарь хочет обедать, он ест из горшка один кусок, если только горшок не пуст, иначе дикарь будит повара и ждет, пока тот не наполнит горшок. Повар, сварив обед, засыпает. Создать приложение, моделирующее обед дикарей. Каждый дикарь должен быть представлен отдельным процессом.
